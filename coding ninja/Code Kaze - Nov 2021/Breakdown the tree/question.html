Ninja has decided to conduct a two-day-long marathon and is still confused regarding the path. He comes up with a plan but is not sure whether he can implement it and is asking for your help.

He loves Data Structures and has decided to create a path following the concept of “trees.” The path would consist of ‘N’ stops where food and water will be available and the path between any 2 stops will be considered as the tree’s edge. Now, he wants to breakdown his journey into simpler paths. He describes the breakdown as the process of dividing the tree's edges into some simple pathways, each with at least one common vertex. The tree's edges should all follow the same course.

Can you help him design his marathon route?

Input format:
The first line contains an integer N determining the number of stops in the marathon route.
The next N-1 lines contain 2 integers, Xi and Yi, the stops that are connected to each other by a common road.
Output format:
If it is not possible to construct the route, print “NO”.
Otherwise print “YES”, and in the next line, print the total number of paths in this marathon route, say X.
Constraints
2 ≤ N ≤ 10^5
1 ≤ Xi, Yi ≤ N
